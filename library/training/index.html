Posts

If you’ve used Wordpress before, adding posts is the same as you’re used to. From anywhere on the site using the header choose New > Post or from the backend navigate to Posts on the left-hand-side sidebar and click New Post.

Editing Content

When editing content, there are a few things to keep in mind to ensure the post looks great and is readable to search engines.

Headers

These are H1, H2, H3, H4, H5 & H6

Headers describe hierarchy to search engines, so an H1 provides canonical information that’s more important than H2-H6, as a result there should only be one H1 per page or post. For posts, the title is the H1 so you’ll never need to add an H1 to a post.

For example of the heirachry mentioned: this document. The title is Chrisa Pappas Training and that’s an H1. You’ll see the title Posts, (an H2) within which all content has to do with posts — there’s a title Editing Content and that’s an H3. Finally, you get to the Headers section, which is the one we’re in now and that’s an H4. Content is separated in a way that gives context to a reader and to a search engine, and the size/style of the headers give a visual cue to readers as to how things are organized.

Since headers have predefined styles, don’t ad any Italic or Bold styles to them.

Images

Featured Image

This is the image on the right-hand side of the post-editor page, feel free to upload a large image, the server will automatically resize it when you load the page. Since this is used in quite a few places, be sure to upload an image here.

Custom Images

There are two custom image sizes that you can optimize to look their best:

  • The Grid Image - 800px x 920px. This is the image you see when there’s a grid of posts (homepage, post pages etc.
  • The Pinned Post Image - 1260px x 860px. This is the image that displays on the pinned post section on the homepage. Even if you’re not going to pin a post, you should upload this image because it can still be displayed if it’s the most recent post.

You don’t need to upload these—we re-size the featured image if they’re not available—but it’s not foolproof, sometimes faces will be cropped since it’s programmatically cropping.

Content Images

When uploading images, Wordpress conveniently re-sizes them so you can display different images at different screen resolutions. The image size that you’ll want to choose to the content-area (the blog-post itself) is Large - 1380px the height is automatically determined by the image size, so that will be different. This should be selected by default so you most likely won’t need to worry about it. No need to align images to the center, left or right — set them to No Alignment.

Pinning Posts

The pinned post on the homepage is a good way to display content that you want readers to see first for a short time when it would otherwise be swept into the feed behind newer posts.

The pinned post section will display the most recent pinned post that hasn’t expired. If there are no pinned posts to display the site will automatically display the most recent post whether or not it’s a pinned post.

So if you have an event that you would like to have pinned, but would like it to display as a normal post after the event has passed, just choose an expiration date after the event; the post will display in normal chronological order.

Options to pin a post are available while editing a post under Post Fields > Options.

Site Options & Defaults

From the dashboard, the left-hand-side menu item titled Options has a few site-wide settings. Most of them won’t need to be touched, but we like to add them as options for peace-of-mind.

They are as follows:

  1. Social Media
  2. Email Block - This is the signup form that people use, there’s a handful of code that you won’t want to touch but the title and subtitle are editable.
  3. Text - This has a text-field who’s content lives on the search-results page when search can’t find anything.

Menus

The menu is very basic and runs off wordpress defaults, so you’re probably familiar with the way it works. Under the left-hand-side menu Appearance > Menus you’ll see a simple UI to drag-and-drop pages into the navigation.
